# Security Policy

## Supported Versions

| Version       | Supported          |
| ------------- | ----------------- |
| 12.x          | ✅                |
| 11.x and below| ❌                |

## Reporting a Vulnerability

If you discover a security vulnerability, please follow these steps:

1. **Do not open a public issue.**
2. Email us at [yousaf.bsse1735@gmail.com](mailto:yousaf.bsse1735@gmail.com) with details.
3. Include steps to reproduce, potential impact, and any relevant logs or code.

We will respond as quickly as possible and coordinate a fix.

## Disclosure Policy

We ask that you give us a reasonable time to address the issue before public disclosure.

Thank you for helping keep this project secure!
